Сайт автоматически формирует ссылки на страницы пользователей, собирая их из отдельных частей. Соберите ссылку с помощью конкатенации и выведите на экран:

```text
https://github.com/hexlet/exercises-python
```

Каждый компонент URL — отдельная строка: протокол, домен, имя владельца, имя репозитория.
