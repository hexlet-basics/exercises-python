В трёх именах участников спрятано секретное слово. Составьте и выведите его из символов в таком порядке:

1. Второй символ из первой строки
2. Третий символ из третьей строки
3. Третий символ из второй строки
4. Седьмой символ из второй строки
5. Пятый символ из второй строки

Попробуйте использовать интерполяцию (f-строки): внутри фигурных скобок можно помещать не только целые переменные, но и отдельные символы с помощью квадратных скобок.
