В коде заданы данные об автомобильной поездке: расстояние, расход топлива, цена топлива и количество пассажиров.

Вычислите и выведите на экран три значения:

1. Объем топлива в литрах, необходимый для поездки. Округлите до одного знака после запятой.
2. Общую стоимость поездки в рублях. Округлите до двух знаков после запятой.
3. Стоимость поездки для каждого пассажира в рублях. Округлите до целого числа, вызвав `round()` без второго аргумента.

Каждое значение выводится на отдельной строке.

```
distance ──────┐
               ├──→ fuel ──────┐
fuel_consumption ──┘           ├──→ trip_cost ──────┐
                               │                    ├──→ per_person
fuel_price ────────────────────┘                    │
                                                    │
passengers ─────────────────────────────────────────┘
```
