Сайт выводит приветствие для каждого посетителя. Реализуйте функцию `say_hello()`, которая выводит на экран фразу `Hello, World!`.

```python
say_hello() # => Hello, World!
```

В задачах, в которых нужно реализовать функцию, эту функцию вызывать не нужно. Вызывать функцию будут автоматизированные тесты, которые проверяют ее работоспособность. Пример с вызовом выше показан только для того, чтобы вы понимали, как ваша функция будет использоваться.
