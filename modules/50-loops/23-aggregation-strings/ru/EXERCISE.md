Реализуйте функцию `sanitize_phone_number()`, которая принимает номер телефона из формы и возвращает строку без пробелов, скобок и дефисов.

Пользователи вводят номера по-разному, но перед сохранением их часто приводят к одному виду. Пройдите исходную строку посимвольно и соберите новый номер только из полезных символов.

```python
sanitize_phone_number('+7 (999) 123-45-67')  # '+79991234567'
sanitize_phone_number('8 800 555 35 35')     # '88005553535'
sanitize_phone_number('(123) 456-7890')      # '1234567890'
```

### Подсказки

* Итоговый результат - строка.
* Используйте пустую строку как начальное значение.
